# Poom-Pai

## 💸 공동 가계부 서비스 Poom-Pai

### 📢 불만없는 정산 시스템을 만들자

- 회비를 미리 설정 할 수 있습니다. 💡 선입금 기능
- 공평한 정산 금액을 확인하세요 💡 정산 내역 확인 기능
- 총 경비, 총 사용 금액, 남은 금액 확인이 가능합니다.
- 장소 별 사용 금액 작성이 가능합니다.
- 세부 항목 추가 가능
- ❗각 장소별, 장소 세부 항목 별 정산 제외 인원 선택 가능!

## 🔍 주요 기능

- 모임 등록 - 모임 제목, 입장 코드, 수정 코드 설정만 하면 끝!
- 입장 코드로 해당 모임 내역 확인이 가능합니다.
- 모임 내역 저장 가능
- url 공유 시 정산 내역 공유 가능 
- 기본 / 수정 모드 제공 - 수정 시 수정 코드 입력 필요
- 선입금 금액 표시
- 참석자가 받아야 하는 금액 or 더 지출해야하는 금액 표시 (뿜빠이)
- 지출 내역 작성
- 각 지출 내역에 포함되지 않는 인원 제외 기능
- 각 사용 장소 세부 내역 별 인원 제외 가능

## ✍️ 용어 컨벤션

- 모임 제목 : meetTitle
- 입장 코드 : meetEntryCode
- 수정 코드 : meetEditCode

- 유저 ID : userId
- 유저 이름 : userName
- 선입금 : upfrontPayment
- 지출 내역 장소 ID : placeId
- 지출 내역 장소 이름 : placeName: string;
- 지출 내역 장소 총 사용 금액 : placeTotalPrice
- 지출 내역 장소 세부 내역 그릅 : placeDetails
- 지출 내역 별 제외 인원 : placeExcludeUser
    - 세부내역 ID : placeItemId
    - 세부내역 이름 : placeItemName
    - 세부내역 사용 금액 : placeItemPrice
    - 세부내역 제외 인원 : placeItemExcludeUser
